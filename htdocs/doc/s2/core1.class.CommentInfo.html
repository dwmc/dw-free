CommentInfo Class

CommentInfo Class — Information about comments attached to something.

Members

»int count

Current number of comments available to be read by the viewer.

»bool enabled

Set to false if comments disabled journal-wide or just on this item.

»string post_url

(Read-only) URL pointer to the 'Post Comments' view.

»string read_url

(Read-only) URL pointer to the 'Read Comments' view.

»bool screened

Set to true if there are screened comments and remote user can unscreen them.
