LJ::run_hook

LJ::run_hook — Runs single site-specific hook of the given name.

Use

LJ::run_hook(hookname, args);

Arguments

  • hookname. Name of hook. See hooks guide.

  • args. Arguments to be passed to hook.

Source:

cgi-bin/ljhooks.pl

Returns:

return value from hook
