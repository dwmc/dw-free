LJ::set_rel_multi

LJ::set_rel_multi — Sets relationship edges for lists of user tuples.

Use

LJ::set_rel_multi(edges);

Arguments

  • edges. array of arrayrefs of edges to set: [userid, targetid, type]. Where: userid: source userid, or a user hash; targetid: target userid, or a user hash; type: type of the relationship.

Source:

cgi-bin/ljrelation.pl

Returns:

1 if all sets succeeded, otherwise undef
