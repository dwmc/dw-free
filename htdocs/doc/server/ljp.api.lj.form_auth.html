LJ::form_auth

LJ::form_auth — Creates an authentication token to be used later to verify that a form submission came from a particular user.

Use

LJ::form_auth(raw);

Arguments

  • raw. boolean; If true, returns only the token (no HTML).

Source:

cgi-bin/weblib.pl

Returns:

HTML hidden field to be inserted into the output of a page.
