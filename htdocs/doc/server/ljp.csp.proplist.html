Chapter 30. Journal Item Meta-data

This is a list of the meta-data that can be assigned to journal items.

Log Prop List

current_mood

Current Mood. Your current mood.

  • Datatype: char

  • Scope: general

current_moodid

Current Mood ID#. Your current mood ID number, if known.

  • Datatype: num

  • Scope: general

current_music

Current Music. Music you're currently listening to.

  • Datatype: char

  • Scope: general

opt_preformatted

Don't Auto-Format. Turn on if post contains HTML and shouldn't be formatted

  • Datatype: bool

  • Scope: general

opt_nocomments

Don't Allow Comments. Turn on if readers can't post comments on this entry.

  • Datatype: bool

  • Scope: general

picture_keyword

Picture Keyword. A keyword that should align to a defined picture

  • Datatype: char

  • Scope: general

opt_backdated

Back-dated. Set to true if this item shouldn't show up on people's friends lists (because it occurred in the past)

  • Datatype: bool

  • Scope: general

opt_noemail

Don't email comments. Turn on if the poster isn't interested in receiving comments to this post by email

  • Datatype: bool

  • Scope: general

opt_screening

Custom Screening Level. Like opt_whoscreened: A = All, R = Remote needed (anonymous only), F = non-Friends, N = None, else use userprop.

  • Datatype: char

  • Scope: general

commentalter

Comments altered. Unix time of the last change to number of comments to this post.

  • Datatype: num

  • Scope: general

syn_link

Syndication item link URL. Original URL of syndication item

  • Datatype: char

  • Scope: general

unknown8bit

Unknown 8-bit text. True if text has 8-bit data that's not in UTF-8

  • Datatype: bool

  • Scope: general

sms_msgid

SMS Message ID. SMS Message ID of the message which led to this journal entry

  • Datatype: num

  • Scope: general

syn_id

Syndicated item id. Unique id of syndication item

  • Datatype: char

  • Scope: general

revtime

Revision time. Unix time of the last edit

  • Datatype: num

  • Scope: general

revnum

Revision number. Number of times this post has been edited.

  • Datatype: num

  • Scope: general

hasscreened

Has screened replies. True if comments to this item include screened comments

  • Datatype: bool

  • Scope: general

taglist

Tag List. Comma separated list of tags on the entry

  • Datatype: char

  • Scope: general

used_rte

Composed in RTE. True if entry was composed using the rich text editor

  • Datatype: bool

  • Scope: general

current_location

Current Location. Current location at time of post, free form text

  • Datatype: char

  • Scope: general

current_coords

Current Coordinates. Current coordinates at time of post, in form '45.2935N 123.3452W'

  • Datatype: char

  • Scope: general

useragent

User Agent. Name of web/mobile/sip/etc client used to post

  • Datatype: char

  • Scope: general

verticals_list

Verticals List. Comma-separated list of verticals that this entry is in

  • Datatype: char

  • Scope: general

qotdid

Writer's block ID. Writer's block ID

  • Datatype:

  • Scope: general

adult_content

Adult Content Flag. Sets the entry as containing adult content (none, explicit, concepts)

  • Datatype: char

  • Scope: general

personifi_lang

Personifi language. The language which the entry was found by Personifi

  • Datatype: char

  • Scope: local

personifi_tags

Personifi tags. Raw tags which came back from Personifi

  • Datatype: char

  • Scope: local

personifi_word_count

Personifi word count. The word count found to be in the entry by Personifi

  • Datatype: num

  • Scope: local

admin_content_flag

Admin Content Flag. Internal flag describing the nature of this entry, set by an admin

  • Datatype: char

  • Scope: general
