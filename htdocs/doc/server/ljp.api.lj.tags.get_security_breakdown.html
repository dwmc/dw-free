LJ::Tags::get_security_breakdown

LJ::Tags::get_security_breakdown — Returns a list of security levels that apply to the given security information.

Use

LJ::Tags::get_security_breakdown(security, allowmask);

Arguments

  • security. 'private', 'public', or 'usemask'

  • allowmask. a bitmask in standard allowmask form

Source:

cgi-bin/taglib.pl

Returns:

List of broken down security levels to use for logkwsum table.
