LJ::memcache_kill

LJ::memcache_kill — Kills a memcache entry, given a userid and type.

Use

LJ::memcache_kill(uuserid, type);

Arguments

  • uuserid. a userid or u object

  • type. memcached key type, will be used as "$type:$userid"

Source:

cgi-bin/LJ/User.pm

Returns:

results of LJ::MemCache::delete
