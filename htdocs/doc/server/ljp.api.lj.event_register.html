LJ::event_register

LJ::event_register — Logs a subscribable event, if anybody is subscribed to it.

Use

LJ::event_register(dbarg, dbc, etype, ejid, eiarg, duserid, diarg);

Arguments

  • dbarg. Deprecated; a master database handle ($dbh) or a master/slave set of database handles ($dbs).

  • dbc. Cluster master of event

  • etype. One character event type.

  • ejid. Journalid event occurred in.

  • eiarg. 4 byte numeric argument

  • duserid. Event doer's userid

  • diarg. Event's 4 byte numeric argument

Source:

cgi-bin/ljlib.pl

Returns:

boolean; 1 on success; 0 on fail.
