LJ::send_mail

LJ::send_mail — Sends email. Character set will only be used if message is not ASCII.

Use

LJ::send_mail(opt, async_caller);

Arguments

  • opt. Hashref of arguments. Required: to, from, subject, body. Optional: toname, fromname, cc, bcc, charset, wrap, html. Note: charset covers body parts only (FIXME - RFC2047) Warning: opt can be a MIME::Lite ref instead, in which case it is sent as-is.

  • async_caller.
