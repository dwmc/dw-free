LJ::Memories::count

LJ::Memories::count — Returns the number of memories that a user has.

Use

LJ::Memories::count(uuobj);

Arguments

  • uuobj. Userid or user object to count memories of.

Source:

cgi-bin/ljmemories.pl

Returns:

Some number; undef on error.
