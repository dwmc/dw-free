LJ::Schools::approve_pending

LJ::Schools::approve_pending — Takes a bunch of pending rows and approves them as a new target school.

Use

LJ::Schools::approve_pending(pendids, options);

Arguments

  • pendids. Arrayref of pendids from the schools_pending table.

  • options. Hashref; Key=>value pairs that define the target school's information. Keys are one of: name, city, state, country, citycode, statecode, countrycode, url.

Source:

cgi-bin/schoollib.pl

Returns:

Allocated school id on success, Undef on error.
