LJ::userpic_count

LJ::userpic_count — Gets a count of userpics for a given user.

Use

LJ::userpic_count(dbarg, upics, idlist);

Arguments

  • dbarg. Deprecated; a master database handle ($dbh) or a master/slave set of database handles ($dbs).

  • upics. hashref to load pictures into, keys being the picids

  • idlist. [$u, $picid] or [[$u, $picid], [$u, $picid], +] objects also supports deprecated old method, of an array ref of picids.
