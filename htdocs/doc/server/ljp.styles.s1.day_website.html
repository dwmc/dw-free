DAY_WEBSITE

DAY_WEBSITE — When the user has specified their website URL, this variable goes into the %%website%% property of DAY_PAGE.

View Types:

day

Overrideable:

Yes; users of this style may override this

Variable Type

HTML

Properties

url The URL of the user's website.
name The name of the user's website, or "my website" if they haven't specified it.
