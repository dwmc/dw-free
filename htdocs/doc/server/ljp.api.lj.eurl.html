LJ::eurl

LJ::eurl — Escapes a value before it can be put in a URL. See also LJ::durl.

Use

LJ::eurl(string);

Arguments

  • string. string to be escaped

Source:

cgi-bin/ljtextutil.pl

Returns:

string escaped
