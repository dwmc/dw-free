2. A list of all Privs

The following is a complete privilege reference.

User Privileges

accountsexpiring -- (scope: local)

Accounts - Expiring. Allows a user to view paid accounts that are going to expire soon

admin

Administer privileges. Allows a user to grant or revoke privileges to/from other users.

Argument: Unique privcode that the user has access to grant/deny for, or "*" for all privileges.

allowopenproxy

Allow Open Proxy. Allows a user to set an IP address as being allowed if it's listed as an open proxy.

betatest

Beta Test Features. Allows a user to access new features that are under development.

Argument: Feature codename

canview

View All Entries. Allows a user to view information that isn't otherwise available. All use is logged.

Argument: Arg="*": can view everything, Arg="suspended": can view public posts in a suspended journal, Arg="styles": can see private styles.

changejournaltype

Change Journal Type. Allows a user to change another user's journal type.

communityxfer

Community Maintainer Transfer. Allows a user to change the maintainer of a community.

contrib_delete -- (scope: local)

Delete Contributions. Allows a user to delete contributions from idiots.

contrib_edit -- (scope: local)

Edit Contributions. Allows a user to edit contributions.

deletetalk

Delete Comments or Entries. Allows a user to delete comments or entries from other journals. Only used in rare cases.

dirsync

Directory Sync. Allows a user to FTP files and have them copied over to the main site's document roots. An example argument would be: "client/win32=files/bin/clients/win32".

Argument: Unique path of the form "devftppath=htdocpath"

faqadd

FAQ - Add. Allows a user to add documents to the Frequently Asked Questions.

Argument: Unique faq category code that the privilege is applicable for, or "*" for all faq categories.

faqcat

FAQ - Edit Categories. Allows a user to modify FAQ categories.

faqedit

FAQ - Edit. Allows a user to edit documents in the Frequently Asked Questions.

Argument: Unique FAQ category code that the privilege is applicable for, or "*" for all faq categories

fileedit

File Editing. Allows a user to edit a static file in the main site's document include tree.

Argument: Unique file identifier, or "*" for all files

finduser

Find a user. Allows a user to find other users by different criteria.

Argument: Optional "codetrace", which allows the user to see someone's use of account codes.

gencodes

Generate Codes. Allows a user to generate unlimited numbers of invite codes.

grantperm -- (scope: local)

Grant Permanent Account. Allows a user to grant another account a Permanent Account

historyview

View statushistory info. Allows a user to view data in the "statushistory" table for all users.

Argument: Type of "statushistory" entries user can see, or no argument to see everything.

moneyenter -- (scope: local)

Money - Enter Payments. Allows a user to record payments for paid accounts.

moneysearch -- (scope: local)

Money - Search for payments. Allows a user to search for payments entered.

moneyview -- (scope: local)

Money - View Detailed Records. Allows a user to see details about incoming money.

moodthememanager

Mood Themes - Manager. Allows a user to edit mood themes and make themes public.

reset_email

Reset User Email Address. Allows a user to delete all email addresses from an account.

reset_password

Reset User Password. Allows a user to reset the password to an account

schemadoc

Edit Schema Documentation. Allows a user to edit the database schema documentation.

sharedjournal

Shared Journal - Admin. Allows a user to add or remove people's access to post in a shared journal.

Argument: Community/shared journal username that's being administrated, or "*" for all journals.

shipping -- (scope: local)

Shipping Access. Allows a user to access shipping tools.

siteadmin

Administer Site. Allows a user to view/adjust certain critical site settings.

Argument: Unique keyword that user has access to view

sponsordomainadm -- (scope: local)

Sponsors Domain Moderator. Allows user to moderate a sponsors domain.

Argument: Sponsors domain name

supportchangesummary

Support - Change Summaries. Allows a user to change the summary on support requests.

Argument: Unique support category

supportclose

Support - Close Requests. Allows a user to close old support requests that other users haven't closed themselves.

Argument: Unique support category that the user has permission to close in.

supportdelete

Support - Delete Requests. Allows a user to delete duplicate or fake support requests.

supporthelp

Support - Manage Requests. Allows a user to approve screened responses, change request summaries and answer and comment unscreened.

Argument: Unique Support Category

supportmakeinternal

Support - Make Internal Comments. Allows a user to make internal comments on support requests.

Argument: Unique support category

supportmovetouch

Support - Move and Touch Requests. Allows a user to move support requests to different categories and to mark requests as still needing help. The supportmakeinternal privilege is required.

Argument: Unique support category

supportread

Support - Read Requests. Allows a user to read support requests in private categories.

Argument: Unique support category

supportviewinternal

Support - View Internal Comments. Allows a user to view internal comments on support requests.

Argument: Unique support category

supportviewscreened

Support - See Screened Responses. Allows a user to read screened responses on support requests.

Argument: Unique support category

supportviewstocks

View stock answers. Allows a user to make use of the stock answers in a support category.

Argument: Unique support category

suspend

Suspend accounts. Allows a user to suspend or unsuspend user accounts. Used by the Abuse Team.

syn_edit

Edit Syndicated Settings. Allows editing settings of syndicated journal that shouldn't be editable by users.

sysban

Modify System Bans. Allows a user to modify bans with the sysban mechanism.

Argument: A specific ban type the user can modify, or "*" for all ban type.

topicaddtopic

Topic Dir - Add Topics. Allows a user to add/approve/deny new topics within a category.

Argument: Unique category number that the user has access in, or "*" for all categories.

topicmanager

Topic Dir - Manager. Allows a user to have all access to administrative functions of the topic directory. They can then create/edit topics, create/edit topic categories, etc.

topicscreencat

Topic Dir - Screen Submissions. Allows a user to approve or deny entries that are submitted to the directory.

Argument: Unique category number that the user has access in, or "*" for all categories

translate

Translate/Update Text. Allows a user to edit site text in a given language.

Argument: Unique language code, optionally appended by |domainid.domaincode

vertical

Moderate Vertical. Allows a user to add and remove entries from a vertical.

Argument: The vertical name, or "*" for all verticals
