LJ::register_setter

LJ::register_setter — Installs code to run for the "set" command in the console.

Use

LJ::register_setter(key, subref);

Arguments

  • key. Key to set.

  • subref. Subroutine reference to run later.

Info

Setters can be general or site-specific.
