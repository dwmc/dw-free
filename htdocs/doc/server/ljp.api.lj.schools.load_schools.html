LJ::Schools::load_schools

LJ::Schools::load_schools — Returns detailed information about schools.

Use

LJ::Schools::load_schools(schoolids);

Arguments

  • schoolids. List of school ids to return.

Source:

cgi-bin/schoollib.pl

Returns:

Hashref; key being schoolid, value being a hashref with keys name, url, citycode, countrycode, statecode.
