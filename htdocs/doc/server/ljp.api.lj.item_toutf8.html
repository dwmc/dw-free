LJ::item_toutf8

LJ::item_toutf8 — convert one item's subject, text and props to UTF-8. item can be an entry or a comment (in which cases props can be left empty, since there are no 8bit talkprops).

Use

LJ::item_toutf8(u, subject, text, props);

Arguments

  • u. user hashref of the journal's owner

  • subject. ref to the item's subject

  • text. ref to the item's text

  • props. hashref of the item's props

Source:

cgi-bin/LJ/Entry.pm

Returns:

nothing.
