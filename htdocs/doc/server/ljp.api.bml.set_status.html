BML::set_status

BML::set_status — Takes a number to indicate a status (e.g. 404, 403, 410, 500, etc.) and sets that to be returned to the client when the request finishes.

Use

BML::set_status(status);

Arguments

  • status. 

Source:

cgi-bin/Apache/BML.pm

Returns:

nothing
