LJ::Support::get_touch_supportlogs_by_user_and_date

LJ::Support::get_touch_supportlogs_by_user_and_date — Get all touch (non-req) supportlogs based on User ID and Date Range.

Use

LJ::Support::get_touch_supportlogs_by_user_and_date(userid, date1, date2);

Arguments

  • userid. User ID to filter on, or Undef for all users.

  • date1. YYYY-MM-DD of beginning date of range

  • date2. YYYY-MM-DD of ending date of range

Source:

cgi-bin/supportlib.pl

Returns:

Support HashRef of Support Logs Array, sorted by log time.
