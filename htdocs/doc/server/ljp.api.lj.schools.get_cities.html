LJ::Schools::get_cities

LJ::Schools::get_cities — Gets information about what cities have been populated with schools.

Use

LJ::Schools::get_cities(countrycode, statecode);

Arguments

  • countrycode. The country code provided from LJ::Schools::get_countries.

  • statecode. The state code provided from LJ::Schools::get_states.

Source:

cgi-bin/schoollib.pl

Returns:

Hashref; citycode as key, cityname as the values.
