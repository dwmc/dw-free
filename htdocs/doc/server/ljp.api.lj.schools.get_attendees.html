LJ::Schools::get_attendees

LJ::Schools::get_attendees — Gets a list of users that attended a school.

Use

LJ::Schools::get_attendees(schoolid, year);

Arguments

  • schoolid. School id to get attendees for.

  • year. Optional; if provided, returns people that attended in this year.

Source:

cgi-bin/schoollib.pl

Returns:

List of userids that attended.
