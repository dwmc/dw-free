LJ::color_todb

LJ::color_todb — Takes an #rrggbb value and returns a 24-bit decimal number.

Use

LJ::color_todb(color);

Arguments

  • color. scalar; an #rrggbb string.

Source:

cgi-bin/ljlib.pl

Returns:

undef if bogus color, else scalar; 24-bit decimal number, can be up to 8 chars wide as a string.
