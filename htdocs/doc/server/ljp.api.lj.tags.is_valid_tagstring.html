LJ::Tags::is_valid_tagstring

LJ::Tags::is_valid_tagstring — Determines if a string contains a valid list of tags.

Use

LJ::Tags::is_valid_tagstring(tagstring, listref, opts);

Arguments

  • tagstring. Opaque tag string provided by the user.

  • listref. If specified, return valid list of canonical tags in arrayref here.

  • opts. currently only 'omit_underscore_check' is recognized

Source:

cgi-bin/taglib.pl

Returns:

1 if list is valid, 0 if not.
