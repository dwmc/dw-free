LJ::reject_pending_member

LJ::reject_pending_member — Rejects someone's request to join a community. Updates authactions and generates an e-mail to the user.

Use

LJ::reject_pending_member(commid, userid);

Arguments

  • commid. userid of the community

  • userid. userid of the user doing the join

Source:

cgi-bin/communitylib.pl

Returns:

1 on success, 0/undef on error
