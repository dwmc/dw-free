LJ::load_rel_user

LJ::load_rel_user — Load user relationship information. Loads all relationships of type 'type' in which user 'userid' participates on the left side (is the source of the relationship).

Use

LJ::load_rel_user(db, userid, type);

Arguments

  • db. 

  • userid. userid or a user hash to load relationship information for.

  • type. type of the relationship

Source:

cgi-bin/ljrelation.pl

Returns:

reference to an array of userids
