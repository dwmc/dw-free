CALENDAR_SKYSCRAPER_AD

CALENDAR_SKYSCRAPER_AD — The HTML tag needed to show a skyscraper ad on your journal.

View Types:

calendar

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

ad The system-provided HTML to print the skyscraper ad.
