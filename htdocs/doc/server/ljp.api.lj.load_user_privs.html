LJ::load_user_privs

LJ::load_user_privs — loads all of the given privs for a given user into a hashref, inside the user record. See also LJ::check_priv.

Use

LJ::load_user_privs(u, priv, arg);

Arguments

  • u. A hashref of a row from the user table, sometimes called a "user object". Although only initially a row from the user table, gets a lot of tacked onto it from various other LJ API functions.

  • priv. Priv names to load (see priv_list).

  • arg. Optional argument. See also LJ::check_priv.

Source:

cgi-bin/LJ/User.pm

Returns:

boolean
