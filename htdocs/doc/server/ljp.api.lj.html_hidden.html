LJ::html_hidden

LJ::html_hidden — Makes the HTML for a hidden form element.

Use

LJ::html_hidden(name, val, opts);

Arguments

  • name. Name of form element (will be HTML escaped).

  • val. Value of form element (will be HTML escaped).

  • opts. Can optionally take arguments that are hashrefs and extract name/value/other standard keys from that. Can also be mixed with the older style key/value array calling format.

Source:

cgi-bin/htmlcontrols.pl

Returns:

HTML
