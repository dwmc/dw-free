LJ::mark_authaction_used

LJ::mark_authaction_used — Marks an authaction as being used.

Use

LJ::mark_authaction_used(aaid);

Arguments

  • aaid. Either an authaction hashref or the id of the authaction to mark used.

Source:

cgi-bin/ljlib.pl

Returns:

1 on success, undef on error.
