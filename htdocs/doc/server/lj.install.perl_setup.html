Abstract

The Perl motto is “There is more than one way to do it”, and the three principles for any programmer are Laziness, Impatience, & Hubris, according to the Perl manual. On top of that, Perl is well-suited for any purpose. Anyway, in this chapter we will briefly cover a Perl installation, and then we will show you which Perl modules are required for a working LiveJournal installation.
