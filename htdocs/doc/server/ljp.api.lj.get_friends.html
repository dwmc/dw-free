LJ::get_friends

LJ::get_friends — Returns friends rows for a given user.

Use

LJ::get_friends(uuserid, mask, memcache_only, force);

Arguments

  • uuserid. a userid or u object.

  • mask. a security mask to filter on.

  • memcache_only. flag, set to only return data from memcache

  • force. flag, set to ignore memcache and always hit DB.

Source:

cgi-bin/LJ/User.pm

Returns:

hashref; keys = friend userids values = hashrefs of 'friends' columns and their values
