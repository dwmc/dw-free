LJ::html_color

LJ::html_color — A text field with attached color preview and button to choose a color.

Use

LJ::html_color(opts);

Arguments

  • opts. Valid options are: 'onchange' argument, which happens when color picker button is clicked, or when focus is changed to text box; 'disabled'; and 'raw' , for unescaped input.

Info

Depends on the client-side Color Picker.

Source:
