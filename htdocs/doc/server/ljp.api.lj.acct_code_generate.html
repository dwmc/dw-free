LJ::acct_code_generate

LJ::acct_code_generate — Creates invitation code(s) from an optional userid for use by anybody.

Use

LJ::acct_code_generate(dbarg, userid, quantity);

Arguments

  • dbarg. Deprecated; a master database handle ($dbh) or a master/slave set of database handles ($dbs).

  • userid. Userid to make the invitation code from, else the code will be from userid 0 (system)

  • quantity. Number of codes to generate (default 1)

Source:

cgi-bin/accountcodes.pl

Returns:

Code generated (if quantity 1), number of codes generated (if quantity>1), or undef on failure.
