LJ::set_lastcomment

LJ::set_lastcomment — Sets the lastcomm memcached key for this user's last comment.

Use

LJ::set_lastcomment(u, remote, dtalkid, life);

Arguments

  • u. Journal they just posted in, either u or userid

  • remote. Remote user

  • dtalkid. Talkid for the comment they just posted

  • life. How long, in seconds, the memcached key should live.

Source:

cgi-bin/weblib.pl

Returns:

undef on failure
