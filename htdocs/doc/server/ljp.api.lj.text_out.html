LJ::text_out

LJ::text_out — force outgoing text into valid UTF-8.

Use

LJ::text_out(text);

Arguments

  • text. reference to text to pass to output. Text if modified in-place.

Source:

cgi-bin/ljtextutil.pl

Returns:

nothing.
