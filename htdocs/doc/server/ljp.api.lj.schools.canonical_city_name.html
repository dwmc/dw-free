LJ::Schools::canonical_city_name

LJ::Schools::canonical_city_name — Canonicalizes a cities name to a standard format.

Use

LJ::Schools::canonical_city_name(city);

Arguments

  • city. Name of the city the school is located in.

Source:

cgi-bin/schoollib.pl

Returns:

Canonicalized name of the city.
