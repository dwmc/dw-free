LJ::text_in

LJ::text_in — do appropriate checks on input text. Should be called on all user-generated text.

Use

LJ::text_in(text);

Arguments

  • text. text to check

Source:

cgi-bin/ljtextutil.pl

Returns:

1 if the text is valid, 0 if not.
