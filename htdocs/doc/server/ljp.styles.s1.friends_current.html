FRIENDS_CURRENT

FRIENDS_CURRENT — When a user has specified "current" information (like current music or current mood), this variable is added to the %%currents%% property of FRIENDS_CURRENTS.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

what Specifies what this is the current item of. It'll be something like "Mood", or "Music". You can change the case with the transforms (%%lc:what%%, etc...).
value The actual current whatever.
