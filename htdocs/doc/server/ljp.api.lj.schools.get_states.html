LJ::Schools::get_states

LJ::Schools::get_states — Gets information about what states have been populated with schools. States and provinces are considered the same thing.

Use

LJ::Schools::get_states(countrycode);

Arguments

  • countrycode. The country code provided from LJ::Schools::get_countries.

Source:

cgi-bin/schoollib.pl

Returns:

Hashref; statecode as key, statename as the values.
