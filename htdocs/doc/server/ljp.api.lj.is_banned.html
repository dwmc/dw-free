LJ::is_banned

LJ::is_banned — Checks to see if a user is banned from a journal.

Use

LJ::is_banned(user, journal);

Arguments

  • user. User hashref or userid.

  • journal. Journal hashref or userid.

Source:

cgi-bin/ljrelation.pl

Returns:

boolean; 1 if "user" is banned from "journal"
