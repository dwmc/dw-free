LJ::Schools::canonical_school_name

LJ::Schools::canonical_school_name — Canonicalizes a school name to a standard format.

Use

LJ::Schools::canonical_school_name(name, city);

Arguments

  • name. Name of the school to canonicalize.

  • city. Name of the city the school is located in.

Source:

cgi-bin/schoollib.pl

Returns:

Canonicalized name of the school.
