LJ::Tags::get_usertagsmulti

LJ::Tags::get_usertagsmulti — Gets a bunch of tags for the specified list of users.

Use

LJ::Tags::get_usertagsmulti(opts, uobj);

Arguments

  • opts. Optional hashref with options. Keys can be 'no_gearman' to skip gearman task dispatching.

  • uobj. One or more user ids or objects to load the tags for.

Source:

cgi-bin/taglib.pl

Returns:

Hashref; { userid => *tagref*, userid => *tagref*, ... } where *tagref* is the return value of LJ::Tags::get_usertags -- undef on failure
