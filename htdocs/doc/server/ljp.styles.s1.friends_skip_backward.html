FRIENDS_SKIP_BACKWARD

FRIENDS_SKIP_BACKWARD — The HTML to make the link to the next (less recent) page of the friends view... going back in time.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

numitems The number of items being displayed on the next page.
url The URL of the page (the HREF of your A tag).
