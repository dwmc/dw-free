LJ::Talk::get_talk2_row_multi

LJ::Talk::get_talk2_row_multi — Gets multiple rows of data from talk2.

Use

LJ::Talk::get_talk2_row_multi(items);

Arguments

  • items. Array of arrayrefs; each arrayref: [ journalu, jtalkid ].

Source:

cgi-bin/talklib.pl

Returns:

Array of hashrefs of row data, or undef on error.
