LJ::clear_rel

LJ::clear_rel — Deletes a relationship between two users or all relationships of a particular type for one user, on either side of the relationship.

Use

LJ::clear_rel(dbs, userid, targetid, type);

Arguments

  • dbs. Deprecated; optional, a master/slave set of database handles.

  • userid. source userid, or a user hash, or '*'

  • targetid. target userid, or a user hash, or '*'

  • type. type of the relationship

Info

One of userid,targetid -- bit not both -- may be '*'. In that case, if, say, userid is '*', then all relationship edges with target equal to targetid and of the specified type are deleted. If both userid and targetid are numbers, just one edge is deleted.

Source:

cgi-bin/ljrelation.pl

Returns:

1 if clear succeeded, otherwise undef
