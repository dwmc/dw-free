You need to make a LiveJournal configuration file. There's an example ljconfig.pl in the doc directory by the name of “ljconfig.pl.txt”. You need to copy this file to etc/ljconfig.pl and then edit it to suit your needs. 

      $ cd /home/lj
      $ cp doc/ljconfig.pl.txt etc/ljconfig.pl

The reason it's not there already is so that future upgrades don't destroy your configuration. The end of ljconfig.pl reads in cgi-bin/ljdefaults.pl so that if we add new options, ljdefaults.pl will be updated and will set sane defaults. After an upgrade you want to glance at ljdefaults.pl (or the original doc/ljconfig.pl.txt) and see if there are any new options you feel like setting.
