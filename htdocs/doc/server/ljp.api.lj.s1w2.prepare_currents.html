LJ::S1w2::prepare_currents

LJ::S1w2::prepare_currents — do all the current music/mood/weather/whatever stuff. only used by ljviews.pl.

Use

LJ::S1w2::prepare_currents(dbarg, args);

Arguments

  • dbarg. Deprecated; a master database handle ($dbh) or a master/slave set of database handles ($dbs).

  • args. hashref with keys: 'entry' (an S2 Entry object), 'vars' hashref with keys being S1 variables and 'prefix' string which is LASTN, DAY, etc.

cgi-bin/ljviews-s1-using-s2.pl
