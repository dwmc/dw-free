LJ::Entry::new

LJ::Entry::new — Gets a journal entry.

Use

LJ::Entry::new(uuserid, opts);

Arguments

  • uuserid. A user id or user object ($u ) to load the entry for.

  • opts. Hash of optional keypairs. 'jitemid' => a journal itemid (no anum) 'ditemid' => display itemid (a jitemid << 8 + anum) 'anum' => the id passed was an ditemid, use the anum to create a proper jitemid.

Source:

cgi-bin/LJ/Entry.pm

Returns:

A new LJ::Entry object. undef on failure.
