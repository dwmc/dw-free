FRIENDS_NEW_DAY

FRIENDS_NEW_DAY — Before any FRIENDS_EVENT that's on a day that no other event has been shown before, this block is inserted into %%events%% in the FRIENDS_EVENTS.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

monshort Month. Short form. Example: Jan, Feb, Mar...
monlong Month. Long form. Example: January, February, March...
yy Year, two digit.
yyyy Year, four digit.
m Month. 1-12.
mm Month. 01-12.
d Day. 1-31.
dd Day. 01-31.
dth Day plus ordinal. 1st, 2nd, 3rd, 4th, 5th...
