LJ::S2::get_tags_text

LJ::S2::get_tags_text — Gets text for display in entry for tags compatibility.

Use

LJ::S2::get_tags_text(ctx, taglistref);

Arguments

  • ctx. Current S2 context

  • taglistref. Arrayref containing "Tag" S2 objects

Source:

cgi-bin/LJ/S2.pm

Returns:

String; can be appended to entry... undef on error (no context, no taglistref)
