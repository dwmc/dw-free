LJ::Memories::delete_by_id

LJ::Memories::delete_by_id — Deletes a bunch of memories by memid.

Use

LJ::Memories::delete_by_id(uuobj, memids);

Arguments

  • uuobj. User id or user object to delete memories of.

  • memids. Arrayref of memids.

Source:

cgi-bin/ljmemories.pl

Returns:

1 on success; undef on error.
