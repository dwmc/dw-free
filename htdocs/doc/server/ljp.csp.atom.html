Chapter 26. Supporting the Atom API

LiveJournal supports the Atom API.

Atom API URI: http://www.livejournal.com/interface/atom/feed

(Formerly, the Atom API URI was http://www.livejournal.com/interface/atomapi/username/feed. This, old, way was redundant: after authentication, the username is already known, and is not needed in the URI.)

Atom API feed discovery URI: http://www.livejournal.com/interface/atom
