LJ::infohistory_add

LJ::infohistory_add — Add a line of text to the table for an account.

Use

LJ::infohistory_add(uuid, what, value, other);

Arguments

  • uuid. User id or user object to insert infohistory for.

  • what. What type of history is being inserted (15 chars max).

  • value. Value for the item (255 chars max).

  • other. Optional. Extra information / notes (30 chars max).

Source:

cgi-bin/LJ/User.pm

Returns:

1 on success, 0 on error.
