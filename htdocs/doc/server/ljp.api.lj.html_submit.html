LJ::html_submit

LJ::html_submit — Makes the HTML for a submit button.

Use

LJ::html_submit(name, val, opts, type);

Arguments

  • name. Name of form element (will be HTML escaped).

  • val. Value of form element, and label of button (will be HTML escaped).

  • opts. Optional hashref of additional tag attributes. A hashref of options. Special options are: 'raw' - inserts value unescaped into select tag; 'disabled' - disables the element; 'noescape' - won't escape key values if set to 1;

  • type. Optional. Value format is type => (submit|reset). Defaults to submit.

Info

If only one argument is given it is assumed LJ::html_submit(undef, 'value') was meant.

Source:

cgi-bin/htmlcontrols.pl

Returns:

HTML
