LJ::register_hook

LJ::register_hook — Installs a site-specific hook.

Use

LJ::register_hook(hookname, subref);

Arguments

  • hookname. Name of hook. See hooks guide.

  • subref. Subroutine reference to run later.

Info

Installing multiple hooks per hookname is valid. They're run later in the order they're registered.
