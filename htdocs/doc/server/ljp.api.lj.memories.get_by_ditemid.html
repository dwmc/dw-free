LJ::Memories::get_by_ditemid

LJ::Memories::get_by_ditemid — Get memory for a given journal entry.

Use

LJ::Memories::get_by_ditemid(uuobj, journalid, ditemid);

Arguments

  • uuobj. User id or user object to get memories for.

  • journalid. Userid for journal entry is in.

  • ditemid. Display itemid of entry.

Source:

cgi-bin/ljmemories.pl

Returns:

Hashref of individual memory.
