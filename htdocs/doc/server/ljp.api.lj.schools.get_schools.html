LJ::Schools::get_schools

LJ::Schools::get_schools — Gets schools defined in a given area.

Use

LJ::Schools::get_schools(countrycode, statecode, citycode);

Arguments

  • countrycode. The country code provided from LJ::Schools::get_countries.

  • statecode. The state code provided from LJ::Schools::get_states.

  • citycode. The city code provided from LJ::Schools::get_cities.

Source:

cgi-bin/schoollib.pl

Returns:

Hashref; schoolid as key, hashref of schools row as value with keys: name, city, state, country, url.
