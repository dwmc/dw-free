LJ::Schools::add_pending_school

LJ::Schools::add_pending_school — Adds a school from a user to the pending list of schools.

Use

LJ::Schools::add_pending_school(uobj, options);

Arguments

  • uobj. User id or object of user that's adding the row.

  • options. Hashref; Key=>value pairs that can include: name, city, state, country, citycode, statecode, countrycode, url.

Source:

cgi-bin/schoollib.pl

Returns:

1 on success, undef on error.
