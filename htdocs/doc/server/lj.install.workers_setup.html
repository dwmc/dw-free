Abstract

Gearman and TheSchwartz distribute jobs, farming out work, to machines that are better-suited to tasks, or to do work in parallel.

As new features are added, it is increasingly likely they will rely on “workers” to handle behind-the-scenes tasks. Although you can set up both technologies to take advantage of a farm of servers, you can run the workers with LiveJournal code on a single machine.
