1. Removing hard-coded English text

Avoiding hard-coded English text in files, and removing any inside existing files is important. There are two reasons why:

  1. So LiveJournal is available for users in other languages [14].

  2. So sites like DeadJournal.com can customize more easily, while still running the latest code.


[14] The site administrator needs to make the language available for translators and end-users, using the @LJ::LANGS config. variable.
