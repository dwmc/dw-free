LJ::Faq::lang_update_in_place

LJ::Faq::lang_update_in_place — Fill in question, summary and answer from database for one or more FAQs.

Use

LJ::Faq::lang_update_in_place(lang, faqs);

Arguments

  • lang. Language to fetch strings for (as a class method).

  • faqs. Array of LJ::Faq objects to fetch strings for (as a class method).

Info

May be called either as a class method or an object method, ie: - $self->lang_update_in_place; - LJ::Faq->lang_update_in_place($lang, @faqs);

Source:

cgi-bin/LJ/Faq.pm

Returns:

True value if successful.
