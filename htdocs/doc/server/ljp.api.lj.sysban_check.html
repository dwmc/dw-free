LJ::sysban_check

LJ::sysban_check — Given a 'what' and 'value', checks to see if a ban exists.

Use

LJ::sysban_check(what, value);

Arguments

  • what. The ban type

  • value. The value which triggers the ban

Source:

cgi-bin/sysban.pl

Returns:

1 if a ban exists, 0 otherwise
