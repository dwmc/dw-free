LJ::Support::get_support_by_daterange

LJ::Support::get_support_by_daterange — Get all the support rows based on a date range.

Use

LJ::Support::get_support_by_daterange(date1, date2);

Arguments

  • date1. YYYY-MM-DD of beginning date of range

  • date2. YYYY-MM-DD of ending date of range

Source:

cgi-bin/supportlib.pl

Returns:

HashRef of support rows by support id
