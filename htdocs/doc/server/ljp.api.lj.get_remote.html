LJ::get_remote

LJ::get_remote — authenticates the user at the remote end based on their cookies and returns a hashref representing them.

Use

LJ::get_remote(opts);

Arguments

  • opts. 'criterr': scalar ref to set critical error flag. if set, caller should stop processing whatever it's doing and complain about an invalid login with a link to the logout page. 'ignore_ip': ignore IP address of remote for IP-bound sessions

Source:

cgi-bin/LJ/User.pm

Returns:

hashref containing 'user' and 'userid' if valid user, else undef.
