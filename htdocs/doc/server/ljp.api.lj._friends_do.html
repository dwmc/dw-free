LJ::_friends_do

LJ::_friends_do — Runs given SQL, then deletes the given userid's friends from memcache.

Use

LJ::_friends_do(uuserid, sql, args);

Arguments

  • uuserid. a userid or u object

  • sql. SQL to run via $dbh->do()

  • args. a list of arguments to pass use via: $dbh->do($sql, undef, @args)

Source:

cgi-bin/LJ/User.pm

Returns:

return false on error
