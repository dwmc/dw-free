LJ::S1::get_themeid

LJ::S1::get_themeid — Loads or returns cached version of given color theme data.

Use

LJ::S1::get_themeid(dbarg, themeid);

Arguments

  • dbarg. Deprecated; a master database handle ($dbh) or a master/slave set of database handles ($dbs).

  • themeid. S1 themeid.

Source:

cgi-bin/ljviews.pl

Returns:

Hashref with color names as keys
