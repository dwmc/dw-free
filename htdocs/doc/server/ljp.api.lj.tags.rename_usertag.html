LJ::Tags::rename_usertag

LJ::Tags::rename_usertag — Deletes a tag for a user, and all mappings.

Use

LJ::Tags::rename_usertag(uobj, type, tag, newname);

Arguments

  • uobj. User object to delete tag on.

  • type. Either 'id' or 'name', indicating the type of the third parameter.

  • tag. If type is 'id', this is the tag id (kwid). If type is 'name', this is the name of the tag that we want to rename for the user.

  • newname. The new name of this tag.

Source:

cgi-bin/taglib.pl

Returns:

undef on error, 1 for success, 0 for tag not found
