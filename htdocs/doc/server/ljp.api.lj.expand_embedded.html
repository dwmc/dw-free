LJ::expand_embedded

LJ::expand_embedded — Used for expanding embedded content like polls, for entries.

Use

LJ::expand_embedded(u, ditemid, remote, eventref, opts);

Arguments

  • u. A hashref of a row from the user table, sometimes called a "user object". Although only initially a row from the user table, gets a lot of tacked onto it from various other LJ API functions.

  • ditemid. 

  • remote. Hashref of remote user containing \'user\' and \'userid\' keys as obtained from LJ::get_remote.

  • eventref. 

  • opts. 

Info

The u-object of the journal in question transmits to the function and its hooks.
