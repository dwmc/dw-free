LJ::Tags::get_usertags

LJ::Tags::get_usertags — Returns the tags that a user has defined for their account.

Use

LJ::Tags::get_usertags(uobj, opts);

Arguments

  • uobj. User object to get tags for.

  • opts. Optional hashref; key can be 'remote' to filter tags to only ones that remote can see

Source:

cgi-bin/taglib.pl

Returns:

Hashref; key being tag id, value being a large hashref (FIXME: document)
