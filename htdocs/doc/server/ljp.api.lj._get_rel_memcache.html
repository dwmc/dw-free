LJ::_get_rel_memcache

LJ::_get_rel_memcache — Helper function: returns memcached value for a given (userid, targetid, type) triple, if valid.

Use

LJ::_get_rel_memcache(userid, targetid, type);

Arguments

  • userid. source userid, nonzero

  • targetid. target userid, nonzero

  • type. type (reluser) or typeid (rel2) of the relationship

Source:

cgi-bin/ljrelation.pl

Returns:

undef on failure, 0 or 1 depending on edge existence
