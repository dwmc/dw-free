LJ::Tags::set_usertag_display

LJ::Tags::set_usertag_display — Set the display bool for a tag.

Use

LJ::Tags::set_usertag_display(uobj, vartype, var, val);

Arguments

  • uobj. User id or object of account to edit tag on

  • vartype. Either 'id' or 'name'; indicating what the next parameter is

  • var. If vartype is 'id', this is the tag (keyword) id; else, it's the tag/keyword itself

  • val. 1/0; whether to turn the display flag on or off

Source:

cgi-bin/taglib.pl

Returns:

1 on success, undef on error
