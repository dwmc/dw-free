LJ::Schools::delete_attended

LJ::Schools::delete_attended — Removes an attended edge from a user/school.

Use

LJ::Schools::delete_attended(uobj, schoolid);

Arguments

  • uobj. User id or object of user doing the attending.

  • schoolid. School id of school being un-attended.

Source:

cgi-bin/schoollib.pl

Returns:

1 on success, undef on error.
