LJ::Memories::create

LJ::Memories::create — Create a new memory for a user.

Use

LJ::Memories::create(uuobj, opts, kwids);

Arguments

  • uuobj. User id or user object to insert memory for.

  • opts. Hashref of options that define the memory; keys = journalid, ditemid, des, security.

  • kwids. Optional; arrayref of keyword ids to categorize this memory under.

Source:

cgi-bin/ljmemories.pl

Returns:

1 on success, undef on error
