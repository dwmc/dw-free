LJ::time_to_cookie

LJ::time_to_cookie — Converts Unix time to format expected in a Set-Cookie header.

Use

LJ::time_to_cookie(time);

Arguments

  • time. unix time

Source:

cgi-bin/ljtimeutil.pl

Returns:

string; Date/Time in format expected by cookie.
