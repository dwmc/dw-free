LJ::get_recommended_communities

LJ::get_recommended_communities — Get communities associated with a user.

Use

LJ::get_recommended_communities(user, types);

Arguments

  • user. 

  • types. The default value for type is 'normal', which indicates a community

Source:

cgi-bin/ljlib.pl

Returns:

array of communities
