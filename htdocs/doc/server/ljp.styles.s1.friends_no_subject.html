FRIENDS_NO_SUBJECT

FRIENDS_NO_SUBJECT — When a journal entry does NOT have a subject, this variable is inserted into the %%subject%% property of FRIENDS_EVENT.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

friend Friend's user name.
name Friend's name.
