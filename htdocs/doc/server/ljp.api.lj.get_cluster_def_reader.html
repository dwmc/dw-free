LJ::get_cluster_def_reader

LJ::get_cluster_def_reader — Returns a definitive cluster reader for a given user, used when the caller wants the master handle, but will only use it to read.

Use

LJ::get_cluster_def_reader(uarg);

Arguments

  • uarg. Either a clusterid scalar or a user object.

Source:

cgi-bin/ljdb.pl

Returns:

DB handle. Or undef if definitive reader is unavailable.
