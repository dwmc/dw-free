LJ::load_mood_theme

LJ::load_mood_theme — Loads and caches a mood theme, or returns immediately if already loaded.

Use

LJ::load_mood_theme(dbarg, themeid);

Arguments

  • dbarg. Deprecated; a master database handle ($dbh) or a master/slave set of database handles ($dbs).

  • themeid. the mood theme ID to load.

Source:

cgi-bin/ljmood.pl
