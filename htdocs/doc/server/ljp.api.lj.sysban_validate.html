LJ::sysban_validate

LJ::sysban_validate — determines whether a sysban can be added for a given value.

Use

LJ::sysban_validate(type, value);

Arguments

  • type. the sysban type we're checking

  • value. the value we're checking

Source:

cgi-bin/sysban.pl

Returns:

nothing on success, error message on failure
