FRIENDS_EVENTS

FRIENDS_EVENTS — A container for a bunch of FRIENDS_EVENT variables. Gets inserted into %%events%% of FRIENDS_PAGE.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

events The actual 'n' number of most recent events from the user's friends' journals. Each individual event is formatted according to the FRIENDS_EVENT variable. Before and after unique days, FRIENDS_NEW_DAY and FRIENDS_END_DAY variables are inserted as well.
