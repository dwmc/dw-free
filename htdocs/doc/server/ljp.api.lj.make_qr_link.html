LJ::make_qr_link

LJ::make_qr_link — Creates the link to toggle the QR reply form or if JavaScript is not enabled, then forwards the user through to replyurl.

Use

LJ::make_qr_link(dtid, basesubject, linktext, replyurl);

Arguments

  • dtid. dtalkid for this comment

  • basesubject. parent comment's subject

  • linktext. text for the user to click

  • replyurl. URL to forward user to if their browser does not support QR.

Source:

cgi-bin/weblib.pl

Returns:

undef upon failure or HTML for the link
