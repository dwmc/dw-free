LJ::prepare_currents

LJ::prepare_currents — do all the current music/mood/weather/whatever stuff. only used by ljviews.pl.

Use

LJ::prepare_currents(dbarg, args);

Arguments

  • dbarg. Deprecated; a master database handle ($dbh) or a master/slave set of database handles ($dbs).

  • args. hashref with keys: 'props' (a hashref with itemid keys), 'vars' hashref with keys being S1 variables.

Source:

cgi-bin/ljviews.pl
