LJ::check_email

LJ::check_email — checks for and rejects bogus e-mail addresses.

Use

LJ::check_email(void);

Info

Checks that the address is of the form username@some.domain, does not contain invalid characters. in the username, is a valid domain. Also checks for mis-spellings of common webmail providers, and web addresses instead of an e-mail address.

Source:

cgi-bin/emailcheck.pl

Returns:

nothing on success, or error with error message if invalid/bogus e-mail address
