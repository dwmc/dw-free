FRIENDS_ALTPOSTER

FRIENDS_ALTPOSTER — When a user other than the journal owner posts a top-level journal entry in a journal, this variable is inserted into the %%altposter%% property of FRIENDS_EVENT.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

poster The username that's posting.
owner The username that owns the journal (the friend's username).
bgcolor The background color of the friend.
fgcolor The foreground color of the friend.
