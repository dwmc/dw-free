LASTN_RANGE_HISTORY

LASTN_RANGE_HISTORY — This should be the text that says the user is not viewing the most recent entries, but is instead viewing back into the history.

View Types:

lastn

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

numitems The number of items being displayed.
skip The number of items that were skipped to get to this point in the history.
