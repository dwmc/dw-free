LJ::get_urls

LJ::get_urls — Returns a list of all referenced URLs from a string.

Use

LJ::get_urls(text);

Arguments

  • text. Text from which to return extra URLs.

Source:

cgi-bin/ljlib.pl

Returns:

list of URLs
