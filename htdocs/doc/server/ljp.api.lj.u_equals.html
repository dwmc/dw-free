LJ::u_equals

LJ::u_equals — Compares two user objects to see if they are the same user.

Use

LJ::u_equals(userobj1, userobj2);

Arguments

  • userobj1. First user to compare.

  • userobj2. Second user to compare.

Source:

cgi-bin/LJ/User.pm

Returns:

Boolean, true if userobj1 and userobj2 are defined and have equal userids.
