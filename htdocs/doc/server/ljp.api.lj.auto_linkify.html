LJ::auto_linkify

LJ::auto_linkify — Takes a plain-text string and changes URLs into <a href> tags (auto-linkification).

Use

LJ::auto_linkify(str);

Arguments

  • str. The string to perform auto-linkification on.

Source:

cgi-bin/weblib.pl

Returns:

The auto-linkified text.
