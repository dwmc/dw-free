LJ::get_community_row

LJ::get_community_row — Gets data relevant to a community such as their membership level and posting access.

Use

LJ::get_community_row(ucommid);

Arguments

  • ucommid. a userid or u object of the community

Source:

cgi-bin/communitylib.pl

Returns:

a hashref with user, userid, name, membership, and postlevel data from the user and community tables; undef if error.
